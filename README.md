# You Are Bot! — решение Kaggle-соревнования

Репозиторий содержит моё решение соревнования **You Are Bot!** на Kaggle.

Задача — определить, является ли каждый из двух участников диалога ботом или человеком.
Для каждого диалога модель предсказывает две вероятности:

- вероятность, что `participant 0` — бот;
- вероятность, что `participant 1` — бот.

## Идея решения

Решение построено как ансамбль моделей, который объединяет:

1. ручные признаки поведения участников;
2. текстовые признаки на основе TF-IDF;
3. stacking-модель поверх вероятностей базовых моделей.

## Используемые признаки

Для каждого участника диалога извлекаются признаки:

- количество сообщений;
- средняя длина сообщений;
- количество emoji;
- доля сообщений с emoji;
- частота символов `?` и `!`;
- доля сообщений, начинающихся с заглавной буквы;
- доля заглавных букв в тексте;
- средняя длина слов;
- среднее количество слов в сообщении;
- индекс повторения предыдущего сообщения.

## Модели

В решении используются две бинарные задачи:

- `participant 0` — бот или человек;
- `participant 1` — бот или человек.

Базовые модели:

- LightGBM;
- XGBoost;
- CatBoost;
- Random Forest;
- Logistic Regression.

Дополнительно используется TF-IDF модель:

- character-level TF-IDF;
- n-граммы символов от 2 до 3;
- Logistic Regression.

Финальная модель — stacking на out-of-fold вероятностях базовых моделей.
